## Modo de ejecución
Ejecutar en la terminal
```shell
cmake -B build
```
para crear la carpeta de build, y luego 
```
cmake --build build
```
para compilar el ejecutable.

Finalmente, para ejecutar el programa simplemente se usa
```shell
./build/Tarea1
```
